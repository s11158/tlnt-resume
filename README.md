# TLNT.AE · Resume Formatter

Веб-приложение, которое превращает **любое** загруженное резюме (PDF, Word/DOCX, TXT или фото — PNG/JPG) в **единый брендированный PDF** агентства **TLNT.AE** в лаконичном дизайне (тёплый беж + графит).

## Что делает
- Извлекает текст из PDF (pdf.js), DOCX (mammoth.js), TXT и изображений (OCR через tesseract.js, рус+англ+араб).
- Автоматически **убирает упоминания источников** (LinkedIn, Indeed, hh.ru, ContactOut, RocketReach, Glassdoor и десятков других) и водяные знаки вида «Powered by…», «Page 1 of 2», «Generated by…».
- Разбирает резюме на разделы (имя, должность, контакты, опыт, образование, навыки и т. д.) — всё можно поправить вручную.
- Генерирует чистый PDF с логотипом TLNT.AE (текстовый по умолчанию; можно загрузить свой PNG/SVG).
- Поддерживает **кириллицу** (встроенный шрифт Roboto).

## Приватность
Всё работает **локально в браузере** — файлы никуда не загружаются. Идеально для персональных данных кандидатов.

## Запуск локально
Просто откройте `index.html` в браузере (нужен интернет — библиотеки и шрифт берутся с CDN).
